# E-Commerce API

A complete E-Commerce API built with Node.js, Express, MongoDB, and OAuth authentication.

## Features

- ✅ **Four Collections**: Products, Reviews, Users, Orders
- ✅ **Full CRUD Operations**: GET, POST, PUT, DELETE for all collections
- ✅ **Data Validation**: Joi validation with custom error messages
- ✅ **OAuth Authentication**: Google OAuth 2.0 with JWT tokens
- ✅ **Protected Routes**: Role-based authorization
- ✅ **Comprehensive Testing**: Jest unit tests with Supertest
- ✅ **API Documentation**: Swagger/OpenAPI documentation
- ✅ **Error Handling**: Centralized error handling middleware
- ✅ **Security**: Helmet, CORS, rate limiting, password hashing
- ✅ **Pagination & Filtering**: For product listings
- ✅ **MongoDB Integration**: With Mongoose ODM

## Prerequisites

- Node.js 14+
- MongoDB (local or Atlas)
- Google OAuth credentials
